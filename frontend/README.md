# Evidence-First Authoring System

A prototype for evidence-based legal document authoring, designed for EB-1A petition writing.

一个基于证据优先的法律文书写作原型系统，专为 EB-1A 移民申请设计。

## Features / 功能

- **4 Visualization Modes** - Line, Embedded, Sankey, Matrix views for evidence mapping
- **Material Organization** - Hierarchical Exhibit/Section structure with drag-and-drop
- **Interactive Mapping** - Connect evidence snippets to legal standards
- **Scroll-aware UI** - Connection lines with boundary clamping

---

- **4 种可视化模式** - 线条、嵌入、桑基图、矩阵视图
- **材料整理** - 层级化的 Exhibit/Section 文件夹结构，支持拖拽
- **交互式映射** - 将证据片段连接到法律标准
- **滚动感知 UI** - 连接线自动钳制到可视边界

## Tech Stack / 技术栈

React + TypeScript + Tailwind CSS + Vite

## Quick Start / 快速开始

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:5173
